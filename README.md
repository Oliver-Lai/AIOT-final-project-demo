# 智慧池塘水質監測系統 Demo

## 📋 專案說明

這是一個池塘水質監測系統的 Demo 應用程式，展示了智慧水產養殖中的即時監測與 AI 分析功能。

## ✨ 功能特色

### 1. 即時數據監測
- **三維度水質指標**：硝酸鹽 (NITRATE)、酸鹼值 (PH)、氨氮 (AMMONIA)
- **動態圖表展示**：使用 Chart.js 實現即時更新的折線圖
- **數據來源**：使用 11-02-2022 00:59 之後的實際池塘數據
- **自動更新**：每 2 秒自動更新一筆新數據，模擬即時監測

### 2. AI 智能分析報告
- **一鍵生成**：點擊按鈕即可生成完整的水質分析報告
- **多維度分析**：
  - 數據摘要統計
  - 主要指標分析（平均值、最高/最低值）
  - 水質狀態評估
  - 智能建議與改善措施
  - 趨勢預測
- **視覺化呈現**：彩色警告提示和結構化報告內容

### 3. 狀態監控面板
- 即時顯示當前監測站點
- 最新時間戳記
- 各項水質指標當前值
- 系統運行狀態指示器

## 🚀 使用方法

### 方法一：直接開啟 HTML 檔案
1. 找到 `pond_monitor_app.html` 檔案
2. 使用任何現代瀏覽器（Chrome、Firefox、Edge 等）開啟
3. 即可看到系統自動開始運行

### 方法二：使用 VS Code Live Server
1. 在 VS Code 中開啟 `pond_monitor_app.html`
2. 右鍵選擇「Open with Live Server」
3. 瀏覽器會自動開啟應用程式

## 📊 數據說明

- **數據來源**：Ponds.csv 檔案中 11-02-2022 00:59 之後的實際監測數據
- **監測站點**：Station 1
- **時間間隔**：原始數據為每 20 分鐘一筆，Demo 中設定為每 2 秒更新一筆
- **循環播放**：數據播放完畢後會自動重新開始

## 🎯 核心技術

- **前端框架**：純 HTML5 + CSS3 + JavaScript
- **圖表庫**：Chart.js 3.x
- **響應式設計**：支援桌面、平板、手機等多種裝置
- **無需後端**：完全在瀏覽器端運行，無需伺服器

## 🔬 模擬 AI 分析功能

Demo 中的 AI 分析報告是基於實際數據計算的：

1. **統計計算**：
   - 計算最近 10 筆數據的平均值
   - 找出最高/最低值
   - 分析數據趨勢

2. **智能判斷**：
   - PH 值 < 5.3：發出警告，建議調整
   - 硝酸鹽 > 25 PPM：建議增加換水
   - 氨氮 > 0.05 mg/l：建議使用益生菌

3. **報告生成**：
   - 延遲 2 秒模擬 AI 處理時間
   - 根據數據動態生成建議內容
   - 提供具體的操作建議

## 🛠️ 技術細節

### 圖表配置
- 三個 Y 軸分別對應三種水質指標
- 最多顯示 15 個時間點的數據
- 平滑曲線效果（tension: 0.4）

### 數據更新機制
```javascript
setInterval(updateData, 2000); // 每 2 秒更新一次
```

### 響應式斷點
- 桌面：雙欄布局（圖表 + 分析）
- 平板 (< 1024px)：單欄布局
- 手機 (< 768px)：垂直排列所有元素

## 📝 注意事項
